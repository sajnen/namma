About Namma Nildana

This page contains details about the Namma Nildana bus stop survey project. You can update this information later as the project evolves.

Project Overview

Namma Nildana is a local accessibility survey for bus stops along the Sarjapur Road corridor. The goal is to identify conditions of existing stops, map missing stops, and note accessibility, signage, and cleanliness.

Purpose

This project aims to make bus stop data easier to visualize and improve decision-making for city planners, commuters, and accessibility advocates.
